# Forge — Ship Products, Not Meetings

> Built for Claude Mythos

50 AI agents that argue, iterate, and ship — your product team without the politics.

## Screenshot

![Forge Landing Page](./screenshot-placeholder.png)

## Stack

- **Next.js 14** — App Router, static export
- **Three.js** via @react-three/fiber + @react-three/drei — orbiting cubes visualization
- **Tailwind CSS** — styling
- **TypeScript** — type safety

## 3D Scene

Agent swarm: 12 small cubes orbiting a central larger cube, each on a slightly different orbit radius and speed — visualizing an AI agent team working around a core product.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Outputs to `./out` (static export ready for Netlify).

## Features

- **Auto-Prioritization** — ROI-ranked feature backlog
- **Agent Teams** — PM, design, engineering, QA agents in parallel
- **Continuous Iteration** — Ships 24/7, no sprint cycles
- **Zero Meetings** — Async, documented, traceable decisions

## Deploy

Static export to Netlify. Set build command: `npm run build`, publish dir: `out`.
